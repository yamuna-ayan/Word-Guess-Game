# Word Guessing Game

## 📌 Project Overview

Word Guessing Game is an interactive web-based game where players try to guess a hidden word by entering letters or words. The game provides feedback on correct and incorrect guesses and tracks the player's progress until they either win or lose.

## ✨ Features

* Random word selection
* User-friendly interface
* Input field for guessing letters/words
* Win and loss notification banners
* Score and attempt tracking
* Restart game functionality
* Responsive design

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript

## 🚀 How to Run the Project

1. Download or clone the repository.
2. Open the project folder.
3. Open `index.html` in any web browser.
4. Start guessing the hidden word and enjoy the game.

## 🎮 How to Play

1. A hidden word is selected randomly.
2. Enter a letter or word in the input field.
3. Correct guesses reveal matching letters.
4. Incorrect guesses reduce the remaining attempts.
5. Guess the complete word before running out of attempts to win.
6. Click the restart button to play again.
